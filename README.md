# Your Framework

> A batteries-included React meta-framework that just works.

## Philosophy

- **Zero configuration** — Sensible defaults for everything
- **One way to do things** — No decision fatigue
- **Production-ready** — Not just prototypes, real apps
- **Type-safe** — Full TypeScript support, great DX

## Packages

| Package | Description |
|---------|-------------|
| `@no-bs-framework/core` | Shared utilities and types |
| `@no-bs-framework/state` | Simple, powerful state management |
| `@no-bs-framework/router` | Type-safe routing with guards |
| `@no-bs-framework/data` | Data fetching with optimistic updates |
| `@no-bs-framework/forms` | Form handling made easy |
| `create-no-bs-framework` | CLI to scaffold new projects |

## Quick Start

```bash
npx create-no-bs-framework my-app
cd my-app
pnpm dev
```

## Development

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run playground app
pnpm playground

# Run tests
pnpm test
```
